To create the Logo images:
1. Download the appropriate Logo image
2. Resize it to 60px Height and whatever width will maintain the aspect ratio
3. Save it as a PNG file with 8-bit colour as "Logo.png"
4. Add it to the project as an embedded resource.

To create the footer images:
1. Download the appropriate image.
2. Resize it to 600px Wide maintaining the aspect ratio.
8. Add it to the project as an embedded resource.
